# MuscleStone

「筋トレで成果を出したい方」に向けた筋トレのメニュー管理や体重管理、トレーニングに関しての相互交流ができるサービスです。

## 機能一覧
- 記事投稿機能
- コメント追加機能
- いいね機能
- フォロー機能
- 認証機能
- ページネーション機能


## 使用技術
- インフラ（AWS/Heroku)
- 画像アップロード（carrierwave）
- ページネーション（kaminari）
- パンくずリスト()
- デザイン（Bootstrap）

## 開発環境
- 言語(Ruby)
- フレームワーク(Ruby on Rails)
- 開発環境(Cloud9)
